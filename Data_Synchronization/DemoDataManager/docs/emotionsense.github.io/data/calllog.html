Call Log

This sensor will query the device's call logs and return an anonymised set of call events.

This sensor requires the 'read contacts' and 'read call log' permissions. The JSON format for a sample from this sensor is:

{
	"userid":"user-id",
	"deviceid":"device-id",
	"senseStartTime":"13:51:07:639 30 06 2015 +0100 GMT+01:00",
	"senseStartTimeMillis":1435668667639,
	"dataType":"CallContentReader",
	"contentList":
	[
		{
			"number":"hash-of-number",
			"date":"1402587637523",
			"local_time_when_sensed":"Thu Jun 12 16:40:37 BST 2014",
			"duration":"duration-in-minutes",
			"type":"missed | outgoing | incoming"
		}
	]
}
