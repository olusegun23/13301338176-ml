$$ x = {-b {b^2-4ac} 2a}. $$

方法三：使用MathJax引擎 大家都看过Stackoverflow上的公式吧，漂亮，其生成的不是图片。这就要用到MathJax引擎，在Markdown中添加MathJax引擎也很简单，

$$x={-b{b^2-4ac}}{2a}$$\(x={-b{b^2-4ac}}{2a}\)

然后，再使用Tex写公式。$$公式$$ 表示行间公式，本来Tex中使用(公式) 表示行内公式，但因为Markdown中 是转义字符，所以在Markdown中输入行内公式使用(公式) ，如下代码：

$$x={-b{b^2-4ac}}{2a}$$\(x={-b{b^2-4ac}}{2a}\)
