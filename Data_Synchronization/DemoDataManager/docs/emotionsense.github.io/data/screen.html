Screen Event

The screen sensor captures on/off screen events.

This sensor does not require any permissions. The JSON format for a sample from this sensor is:

{
	"userid":"test-user-id",
	"deviceid":"test-device-id",
	"senseStartTime":"16:26:22:174 30 06 2015 +0100 GMT+01:00",
	"senseStartTimeMillis":1435677982174,
	"dataType":"Screen",
	"status":"SCREEN_OFF"
}
