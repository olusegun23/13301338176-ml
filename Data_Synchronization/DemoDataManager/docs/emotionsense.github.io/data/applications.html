Application Log

This sensor allows you to query for the apps that have been recently run on the device.

WARNING: As of Android Lollipop, this method is no longer available to third party applications.

This sensor requires the Android get tasks permission. The JSON format for a sample from this sensor is:

{
	"userid":"user-id",
	"deviceid":"device-id",
	"senseStartTime":"13:41:44:919 30 06 2015 +0100 GMT+01:00",
	"senseStartTimeMillis":1435668104919,
	"dataType":"Application",
	"applicationResult":
	[
		{
			"running_activities":"1",
			"num_activities":"1",
			"base_activity":"com.emotionsense.demo.data\/.MainActivity",
			"application_label":"Data Manager Demo"
		}
	],
	"senseCycles":1
}
