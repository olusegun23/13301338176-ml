Step Counter

The step counter sensor returns the number of steps that the device has sensed since the last boot.

This sensor does not require any permissions. The JSON format for a sample from this sensor is:

{
	"userid":"test-user-id",
	"deviceid":"test-device-id",
	"senseStartTime":"16:32:46:720 01 07 2015 +0100 GMT+01:00",
	"senseStartTimeMillis":1435764766720,
	"dataType":"StepCounter",
	"stepCount":5,
	"lastBootMillis":1435761935139,
	"lastBoot":"15:45:35:139 01 07 2015 +0100 GMT+01:00"
}
