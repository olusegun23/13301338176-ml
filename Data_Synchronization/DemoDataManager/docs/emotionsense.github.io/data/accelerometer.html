Accelerometer

The accelerometer is a motion sensor that measures the acceleration of the device.

This sensor does not currently require any specific permission. The JSON format for a sample from this sensor is:

{
	"userid":"user-id",
	"deviceid":"device-id",
	"senseStartTime":"12:42:39:031 30 06 2015 +0100 GMT+01:00",
	"senseStartTimeMillis":1435664559031,
	"dataType":"Accelerometer",
	"xAxis":[0.20949265360832214,0.20949265360832214,...],
	"yAxis":[-0.05626374110579491,-0.05626374110579491,...],
	"zAxis":[9.548674583435059,9.548674583435059,...],
	"sensorTimeStamps":[1435664559191,1435664559191,...],
	"sampleLengthMillis":8000
}
